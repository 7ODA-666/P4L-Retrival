# P4L Retrieval

A futuristic Flask-based starter project for information retrieval learning and preprocessing demos.

## Tech Stack

- Python Flask
- HTML
- Tailwind CSS via CDN
- JavaScript
- Font Awesome via CDN

## Project Structure

- `app.py` — Flask server and routes
- `templates/base.html` — shared layout, navbar, footer, CDN includes
- `templates/index.html` — landing page and preprocessing dashboard
- `static/css/style.css` — custom glassmorphism, glow, and animation styles
- `static/js/app.js` — theme toggle, tabs, and small interactions
- `static/images/` — image assets folder

## Run the app

```powershell
pip install flask
python app.py
```

Open the app in your browser at `http://127.0.0.1:5000`.
