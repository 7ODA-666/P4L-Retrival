Edit Distance

Edit Distance measures the minimum number of operations required to transform one word into another. The allowed operations are insertion, deletion, and substitution.

How the matrix is computed

  1. Compare characters diagonally when they are equal (no extra cost).
  2. Consider insertion from the left cell.
  3. Consider deletion from the top cell.
  4. Pick the minimum value among substitution, insertion, and deletion.

Fixed example: cat to cut

Only one substitution is needed: replace a with u. Minimum Edit Distance = 1

Important note: Edit Distance is ideal when correcting typo-level changes because it captures exact character edits.

Jaccard Coefficient

Jaccard Coefficient measures similarity between two words using shared bigrams (two-character chunks), then compares overlap against the full set (union).

Step-by-step breakdown

  1. Split each word into bigrams.
  2. Find shared bigrams between the two lists.
  3. Compute the union count: Total First + Total Second - Shared.
  4. Divide Shared by Union to get similarity.

Formula

Jaccard Coefficient = Shared Bigrams (Total First + Total Second - Shared)

Fixed example: night and nacht

night bigrams: ["ni", "ig", "gh", "ht"]

nacht bigrams: ["na", "ac", "ch", "ht"]

Shared: ["ht"]

1 / (4 + 4 - 1)

1 / 7 = 0.1428

Similarity: 14.28%

Important note: Jaccard is useful for partial similarity, especially when words share segments but are not close by direct character edits.

Soundex Algorithm

Soundex is a phonetic algorithm used to compare words based on pronunciation rather than exact spelling.

Step-by-step breakdown

  1. Keep the first letter.
  2. Convert remaining letters into Soundex digits.
  3. Remove duplicate consecutive digits.
  4. Remove zeros.
  5. Keep the first letter + first 3 digits.

Soundex letter groups

1: B, F, P, V

2: C, G, J, K, Q, S, X, Z

3: D, T

4: L

5: M, N

6: R

Ignored / 0: A, E, I, O, U, H, W, Y

Fixed example: Robert

Step 1: R

Step 2: R0163

Step 3: Remove duplicates

Step 4: Remove zeros

Final: R163

Robert -> R163

Rupert -> R163

Result: Words sound similar.

Important note: Soundex helps when users type names with different spellings but similar pronunciation.

Why Spelling Correction Matters

Spelling correction improves search engines, NLP systems, typo correction tools, and information retrieval systems by helping users find relevant results even when their queries contain mistakes.
